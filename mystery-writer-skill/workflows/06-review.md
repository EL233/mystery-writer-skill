# 工作流 06：推理小说逻辑校验与文本修订 (Review & Logic Verification)

本工作流旨在指导作者在完成初稿后进入复盘与修订阶段。推理小说的修订是一场严密的“逆向工程”，核心目标是：**修补逻辑漏洞（Plot Holes）、优化线索的公平性、强化误导的隐蔽性，并确保故事在真相大白后具有完美的逆向合理性。**

---

## 一、 复审的核心原则：严苛的读者视角

在进入逐字修改前，必须拉开与作品的距离，以最挑剔的解谜期读者的心态来审视文本：
1.  **线索公平性（Fair Play）：** 确保所有推导真凶的核心线索在最终解答前**均已在文本中明确出现**。绝对不允许在解答篇突然掏出“读者之前绝不可能知道的秘密证据”（机械降神）。
2.  **逆向合理性（Retroactive Logic）：** 站在已知真凶和手法的终点，反向推演全书：凶手在这个场景的行为合理吗？他是否有更简单的选择？他为什么要冒这个险？
3.  **时间线闭环（Timeline Synchronization）：** 所有案发时段、嫌疑人行动、物理移动时间必须精确到分钟，绝不能出现空间或时间上的瞬间移动。

---

## 二、 三轮复审法 (Three-Pass Review Method)

为了避免思维混乱，建议将复审过程分为三轮，每轮只专注一个层面的问题：

### 第一轮：逻辑骨架与诡计校验（The Logic Pass）
*   **物理可行性：** 诡计在现实（或设定的世界观）中是否真的具有操作性？（如：特定机关的传动、毒药的生效时间、特定天气的地理影响）。
*   **动机与心理漏洞：** 凶手大费周章设计这个诡计，其收益是否大于直接杀人？如果存在巨大的心理漏洞（例如：明明可以直接逃跑却要伪造不在场证明反而暴露了自己），必须修正。
*   **排除法的唯一性：** 解答篇的推导是否能**百分之百锁定真凶**？是否存在“其实A嫌疑人换个路线也能做到，只是侦探没提”的漏洞？

### 第二轮：线索分布与信息控制（The Clue Pass）
*   **弱化真线索：** 检查那些一锤定音的真线索，是否在初稿中写得过于扎眼？使用“工作流 05”中的藏匿技术，将其进一步伪装在日常描写、视觉盲区或角色的强烈情绪宣泄中。
*   **强化红鲱鱼（误导）：** 检查读者的注意力在各个阶段是否被成功引开。如果某个“伪解答”破绽太明显，读者会提前猜到真相，必须加固伪解答的逻辑支撑，让它看起来更像“真相”。
*   **删除无效信息（Noise Reduction）：** 删掉那些既不提供氛围、也不作为线索、更不具备误导功能的纯冗余描写。

### 第三轮：文本润色与双重语境（The Prose Pass）
*   **伏笔的“双重语境”微调：** 逐句修改凶手、知情者的对话和神态描写。确保这些句子在“第一遍读（不知道真凶）”时极其自然，而在“第二遍读（知道真凶）”时暴露出完全不同的潜台词。
*   **节奏感优化：** 
    *   压缩繁琐的、纯讲道理的“脑内风暴”和长篇大论的案情陈述。
    *   在案情陷入胶着、文字偏向枯燥时，适时插入嫌疑人之间的冲突、环境氛围的压迫感（如暴风雪、停电）或第二起案件的发生。

## 三、 推理小说专项修稿核对表 (The Ultimate Mystery Checklist)

在修订过程中，请对照以下表格进行地毯式排查：

| 检查维度 | 核心审计问题 | 校验状态 (Y/N) | 修正行动记录 |
| :--- | :--- | :---: | :--- |
| **物理/时间** | 所有关键行动的时间线（Timeline）是否能精确对齐，无物理冲突？ | | |
| **线索公平** | 决定性的那一条线索，是否在全书前 70% 的篇幅内让读者看到过？ | | |
| **误导强度** | “伪解答”是否足够惊艳，能让大部分读者在某一时刻深信不疑？ | | |
| **凶手视角** | 真相揭晓后，凶手之前的每一次出场、每一句发言是否依然站得住脚？ | | |
| **侦探能力** | 侦探看穿真相的瞬间，是否依靠了逻辑或洞察力，而非偶然偷听、巧合？| | |
| **崩坏防御** | 是否存在某个由于作者现代常识疏忽导致的现代漏洞（如：忘了角色有智能手机、有行车记录仪等）？ | | |

## 四、 本阶段产出物与交付标准

完成本工作流后，该项目应达到**可发表/可交付**状态：
1.  **精修稿（Final Polished Draft）：** 逻辑闭环、文字洗练、悬念丛生的最终文本。
2.  **完备时间线与线索对照表（Master Timeline & Clue Map）：** 一份仅供作者备忘的终极表格，详细列出：`时间 | 凶手实际在做什么 | 侦探以为他在做什么 | 文本中埋下了什么线索`。这将是你应对读者“挑刺”和写续作（若有）的最强底牌。
