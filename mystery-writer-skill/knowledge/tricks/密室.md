# 密室 (Locked Room / Impossible Crime)

广义指在物理上、地理上看似绝对无法出入的犯罪现场（如雪地无足迹、孤岛密室）。

## 经典分类（参考约翰·狄克森·卡尔的“密室讲义”）
1. **内部犯罪**：凶手就在密室内部，或在密室发现后假装撞门第一个冲进去顺手杀人/藏匿凶器。
2. **机械机关**：利用线绳、胶带、重力、热胀冷缩等物理装置，在室外完成内部反锁。
3. **心理密室**：现场本不是密室，但通过语言暗示、证人误判，让大家“以为”它在案发时是反锁的。
4. **远程/延迟杀人**：在受害者进入房间前下了慢性毒药，或利用冰块弩箭等在窗外射击后融化。

## 创作要点
1. **结构平面图**：写密室必须明确画出房间的窗、门、家具摆设及通风口，不能凭空多出一个暗道。
2. **越简单越好**：过于复杂的鲁布·戈德堡机械（Rube Goldberg machine）容易让读者感到疲劳且不切实际，高级的密室往往利用心理或极简单的物理小常识。
