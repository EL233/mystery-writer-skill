# 推理小说文风分析与调校 (Tone & Style Analysis)

文风是推理小说的皮肤，直接决定了读者的沉浸度。技能包在生成文本时可参考以下四种典型文风进行基调微调（Fine-tuning）：

## 1. 冷峻写实风格 (Noir / Hard-boiled)
* **特点**：去修饰化、硬朗、多用动词和短句。模仿雷蒙德·钱德勒或松本清张。
* **关键词**：烟草、阴雨、疲惫、冷酷的观察、没有英雄主义。
* **句式示例**：“雨砸在柏油路面上，像无数个细小的耳光。他紧了紧风衣，垃圾桶旁那顶礼帽边缘正往下滴着暗红色的水。他知道，今晚睡不成了。”

## 2. 华丽古典风格 (Gothic / Classical)
* **特点**：充满舞台感、注重环境描写（古宅、黄昏、哥特元素）、语言带有贵族气或学究气。
* **关键词**：阴郁、黄昏、命运的齿轮、钟表声、优雅的恶意。
* **句式示例**：“十角馆的阴影拉得极长，宛如一只多足的黑蜘蛛，将夕阳残存的血色悉数吞噬。空气里弥漫着陈旧木料与海盐混杂的死亡气息。”

## 3. 心理流溢/窒息风格 (Psychological / Internal)
* **特点**：高频使用第一人称、内心独白、重复的神经质问句、对细节的病态放大。
* **关键词**：心跳、耳鸣、黏稠、视线、他们是不是知道了？
* **句式示例**：“第二十七次。他确定自己数了二十七次。隔壁那该死的、规律的皮鞋踱步声。每一步都正好踩在我太阳穴的血管上。他是在等我出去，还是在等我死在这？”

## 4. 严谨逻辑/技术风格 (Procedural / Analytical)
* **特点**：极度客观、克制，甚至带有法医报告或代码般的冰冷感。多数据、多物理方位描述。
* **关键词**：经过测量、误差、呈现出、法医学判定。
* **句式示例**：“尸体呈右侧卧位，尸僵已蔓延至下肢关节。胃内容物呈半消化状态，结合死者最后一次就餐记录，死亡时间应在 22:15 至 22:45 之间，正负误差不超过十五分钟。”
